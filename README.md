# JP Embalagens - Site institucional

Site estático pronto para GitHub, Vercel e domínio próprio.

## Como visualizar localmente
Abra o arquivo `index.html` no navegador.

## Como subir no GitHub
1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Faça commit na branch `main`.

## Como publicar no Vercel
1. Acesse Vercel.com.
2. Clique em **Add New Project**.
3. Importe o repositório do GitHub.
4. Framework: **Other** ou **Static HTML**.
5. Clique em **Deploy**.

## Domínio próprio
Depois do deploy, vá em **Project > Settings > Domains**, adicione o domínio e siga os apontamentos DNS informados pela Vercel.
